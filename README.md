### Setup Steps
python -m venv env <br>
source env/bin/activate <br>
pip install -r requirements.txt <br>
python manage.py migrate <br>
python manage.py runserver <br>
<br>
## Run server

python manage.py makemigrations <br>
python manage.py migrate <br>
python manage.py runserver <br>
